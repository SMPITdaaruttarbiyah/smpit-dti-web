## 🚀 使用指南

### 日常内容管理
1. 登录管理面板
2. 添加/编辑新闻内容
3. 上传相关图片
4. 点击"Sync Now"准备数据
5. 运行`bash manual-sync.sh`部署

### 同步操作
```bash
# 手动同步（推荐）
bash manual-sync.sh

# 测试系统
bash test-admin.sh

# 构建项目
npm run build
```

### 维护操作
```bash
# 清理缓存
npm run clean

# 开发模式
npm run dev

# 生产构建
npm run build
```
