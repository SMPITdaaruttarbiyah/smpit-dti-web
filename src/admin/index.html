üìù Panduan Penggunaan CMS

1. Mengelola Halaman

  • Halaman: Buat atau edit halaman utama website
  • Page Builder: Gunakan 4 tipe blok yang tersedia:
    • Blok Teks: Untuk konten teks biasa
    • Visi & Misi: Otomatis menampilkan data dari school.js
    • Fasilitas: Otomatis menampilkan fasilitas dari school.js
    • Kutipan: Menampilkan kutipan inspiratif dari API

2. Mengelola Berita

  • Berita: Tambahkan berita atau pengumuman terbaru
  • Gambar: Upload gambar untuk setiap berita
  • Tags: Tambahkan tags untuk kategorisasi

3. Pengaturan Sekolah

  • Info Sekolah: Edit data sekolah (nama, alamat, visi, misi)
  • Program & Fasilitas: Kelola program dan fasilitas sekolah
  • Sosial Media: Update link sosial media

4. Workflow

  • Draft: Simpan sebagai draft
  • Review: Ajukan untuk review
  • Publish: Publikasikan ke website
